üìù Integration Examples

1. Auto-initialization with Script Tag

The simplest way to integrate - just add the script tag with data attributes:

<script src="./dist/static/js/omnifeedback-widget.umd.js" data-tenant-id="demo-tenant" data-jwt-token="eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9..." data-api-url="https://api.omnifeedback.ai" data-primary-color="#007bff" data-target="body"> </script>

2. Programmatic Initialization

For more control, initialize the widget programmatically:

<script src="https://unpkg.com/react@19/umd/react.production.min.js"></script> <script src="https://unpkg.com/react-dom@19/umd/react-dom.production.min.js"></script> <script src="https://unpkg.com/styled-components@6/dist/styled-components.min.js"></script> <script src="./dist/static/js/omnifeedback-widget.umd.js"></script> <script> const widget = OmniFeedbackWidget.init({ tenantId: 'your-tenant-id', jwtAuthToken: 'your-jwt-token', apiUrl: 'https://api.omnifeedback.ai', targetElement: '#widget-container', theme: { primaryColor: '#ff6b6b', buttonText: 'Give Feedback' } }); // Control the widget widget.show(); widget.hide(); widget.destroy(); </script>

3. Multiple Widget Instances

You can create multiple widget instances for different purposes:

// Bug reporting widget const bugWidget = OmniFeedbackWidget.init({ tenantId: 'tenant-1', targetElement: '#bug-reports', theme: { primaryColor: '#dc3545' } }); // Improvement suggestions widget const suggestionWidget = OmniFeedbackWidget.init({ tenantId: 'tenant-1', targetElement: '#suggestions', theme: { primaryColor: '#28a745' } });
